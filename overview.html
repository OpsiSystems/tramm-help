The TMS Universe is Tramm's Universe

Tramm covers the full logistics operation — from the moment an order is created to the moment it is delivered. Click on any area in the diagrams below to explore that part of the system.

Planning Phase Execution Phase

Tramm is made up of three interconnected products: the core Tramm TMS platform, the Tramm Mobile app used by drivers and field operators, and Fleetvision for real-time fleet visibility. Together they cover every stage of the logistics operation.

Phase 1

From Order to Trip

Orders come in from clients, get broken down into jobs, loads are built, and trips are planned and allocated to vehicles. This is where the work is organised before it hits the road.

Order Manager → Planner → Planner → Execution Manager →
Order Creation → Order Manager Build Loads → Planner Trip Allocation → Planner Delivery Execution → Execution Manager
Once trips are planned and allocated, they move into the execution phase — drivers receive their routes on Tramm Mobile, and the operations team monitors live progress through Fleetvision.
Phase 2

From Allocation to Delivery

Drivers execute their trips via the Tramm Mobile app — capturing ePODs, managing delivery workflows, and communicating in real time. The operations team monitors everything live through Fleetvision.

Execution Manager → Tramm Mobile → Fleetvision →
Delivery Execution → Execution Manager Tramm Mobile → Tramm Mobile Fleetvision → Fleetvision
